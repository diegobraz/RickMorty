# Rick e Morty
## Description
List of Rick and Morty Series Characters Using Api Public https://rickandmortyapi.com/

## Tech Stack
<ul>
    <li>Kotlin</li>
    <li>MVVM architecture</li>
    <li>Hilt for dependency injection.</li>
    <li>Retrofit</li>
    <li>Coroutines</li>
    <li>Glide</li>
    <li>Modularization</li>
</ul>

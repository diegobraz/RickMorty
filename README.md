# Rick e Morty
## Descrição
Lista de personagens da serie Rick e Morty utilizando Api publica https://rickandmortyapi.com/

## Tech Stack
<ul>
    <li>Kotlin</li>
    <li>MVVM architecture</li>
    <li>Hilt para injeção de dependência.</li>
    <li>Retrofit</li>
    <li>Coroutines</li>
    <li>Glide</li>
</ul>
